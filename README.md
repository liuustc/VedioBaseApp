# 视频库管理工具

一个用于管理本地影视库的Windows桌面应用，支持自动扫描、元数据提取、豆瓣/IMDb评分获取、标签管理、观看状态管理等功能。

## 快速开始

### 启动应用
双击 `start.bat` 即可启动应用。

### 首次使用
1. 点击"选择目录"按钮
2. 选择包含视频文件的目录
3. 程序会自动扫描并提取时长
4. 扫描完成后，电影会显示在列表中

### 后续使用
1. 启动程序后，上次目录会自动加载
2. 点击"刷新列表"按钮重新扫描目录
3. 点击"拉取元数据"获取豆瓣/IMDb评分

## 主要功能

### 影片管理
- **自动扫描**：遍历指定目录，识别视频文件
- **时长提取**：自动提取视频时长（支持moviepy和FFmpeg双方案）
- **下载时间**：显示文件最后修改时间，支持排序
- **播放功能**：详情页一键调用系统默认播放器
- **垃圾文件过滤**：自动识别并过滤带网站水印的垃圾文件

### 评分系统
- **多源评分**：支持豆瓣和IMDb双源评分
- **用户评分**：10分制，支持0.5分刻度
- **评分过滤**：按评分来源（豆瓣/IMDb/我的评分）筛选

### 观看状态
- **状态管理**：未看过 / 已看过 / 想看 / 在看
- **快速切换**：详情页一键切换观看状态
- **状态过滤**：左侧过滤栏按观看状态筛选

### 标签系统
- **自定义标签**：自由创建和管理标签
- **默认标签**：新影片自动标记"未看过"
- **标签计数**：左侧标签栏显示每个标签的影片数量
- **标签过滤**：支持多标签组合筛选

### 列表功能
- **排序支持**：点击表头按任意列排序
- **列宽调整**：支持拖动调整列宽
- **标题优化**：过长标题自动截断，悬停显示完整标题
- **筛选联动**：拉取元数据仅对当前列表生效

- **数据库**：`%USERPROFILE%\.VideoBaseApp\movies.db`
- **配置文件**：`%USERPROFILE%\.VideoBaseApp\config.json`

### 详情页
- 基本信息：标题、文件路径、文件大小、时长
- 评分：豆瓣评分、IMDb评分、我的评分

## 技术栈

- **GUI框架**：PyQt6
- **数据库**：SQLite
- **时长提取**：moviepy / FFmpeg
- **评分API**：OMDb API（IMDb）、豆瓣API

## 常见问题

### Q: 扫描后没有显示时长
A: 点击"刷新列表"按钮，时长会自动提取

### Q: 评分获取失败
A: 豆瓣API可能受限，可尝试切换到IMDb源

### Q: 播放按钮无反应
A: 检查文件是否存在，网络路径需要可访问

### Q: 标签过滤不生效
A: 确保标签已勾选，下拉框选择了正确的状态
