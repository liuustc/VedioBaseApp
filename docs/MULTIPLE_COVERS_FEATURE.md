# 多张封面提取功能

## 功能描述
提取3张不同时间段的视频截图，避免开头和结尾的黑屏问题。

## 实现细节

### 1. metadata.py
- 新增 `extract_multiple_covers()` 方法
- 自动计算视频时长，避免开头和结尾（各留10秒）
- 在中间时间段均匀分布提取3张封面
- 检查文件大小，避免提取到黑屏

### 2. database.py
- 新增 `cover_paths` 字段（JSON格式存储多张封面路径）
- 修改 `add_movie()` 方法支持多张封面

### 3. GUI界面
- 详情对话框显示3张封面（缩略图）
- 扫描时自动提取3张封面

## 封面提取逻辑

1. **获取视频时长**
   - 如果成功：在中间时间段均匀分布提取
   - 如果失败：使用默认时间点（5秒、15秒、30秒）

2. **避免黑屏**
   - 开头留10秒，结尾留10秒
   - 检查文件大小（大于1KB才认为有效）

3. **时间点分布**
   - 视频时长 < 30秒：使用默认时间点
   - 视频时长 >= 30秒：在中间时间段均匀分布

## 使用方法

无需修改，程序会自动：
1. 扫描目录时提取3张封面
2. 详情对话框显示3张封面缩略图
3. 点击封面可查看大图（可选功能）
