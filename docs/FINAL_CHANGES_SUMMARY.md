# 最终修改总结

## 修改1：分离刷新和拉取元数据

### 修改前
- 选择目录后自动扫描、提取元数据、获取评分
- 刷新按钮会重新扫描目录

### 修改后
- **选择目录**：只扫描文件并保存基本信息（文件路径、标题、文件大小）
- **刷新列表**：从数据库加载已有的电影数据
- **拉取元数据**：单独的按钮，用于拉取时长、评分等信息

### 新增功能
- `ScanWorker`：只扫描文件，保存基本信息
- `MetaWorker`：单独拉取元数据（时长、评分）
- `fetch_metadata()`：拉取元数据的方法
- `update_meta_progress()`：更新拉取进度
- `on_meta_finished()`：拉取完成回调
- `on_meta_error()`：拉取错误回调

## 修改2：删除封面相关功能

### 删除的内容

#### metadata.py
- 删除了 `extract_cover()` 方法
- 删除了 `extract_multiple_covers()` 方法
- 删除了 `_resize_cover()` 方法
- 删除了 `cover_dir` 属性
- 只保留了 `get_video_duration()` 和 `check_ffmpeg()`

#### database.py
- 删除了 `cover_path` 字段
- 删除了 `cover_paths` 字段
- 删除了 `_migrate_database()` 方法
- 简化了 `add_movie()` 方法

#### config.py
- 删除了 `get_cover_dir()` 方法
- 删除了 `cover_width` 和 `cover_height` 配置

#### ui/main_window.py
- 删除了封面列显示
- 删除了标签列显示
- 简化了电影列表：ID、标题、文件大小、时长、评分

#### ui/dialogs.py
- 删除了封面显示相关UI
- 删除了封面加载逻辑

## 使用方法

### 扫描目录
1. 点击"选择目录"按钮
2. 选择视频目录
3. 程序只扫描文件并保存基本信息

### 刷新列表
- 点击"刷新列表"按钮
- 从数据库加载已有的电影数据

### 拉取元数据
1. 点击"拉取元数据"按钮
2. 确认后开始拉取
3. 拉取时长、评分等信息
4. 完成后自动刷新列表

## 优点

1. **扫描更快**：不提取元数据，扫描速度更快
2. **按需拉取**：只在需要时拉取元数据
3. **界面简洁**：删除了封面显示，列表更简洁
4. **资源节省**：不提取封面，节省存储空间和处理时间
