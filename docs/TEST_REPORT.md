## 已知问题

1. **豆瓣API限制**：豆瓣有反爬机制，可能返回404错误
   - 解决方案：使用IMDb API作为备选

2. **首次扫描性能**：大目录扫描可能较慢
   - 解决方案：使用多线程扫描（已实现）

3. **封面显示**：GUI中封面显示为文本"有封面"
   - 解决方案：需要实现图片显示功能（可选优化）
