# 时长显示修复

## 问题
时长一栏都是N/A，过滤也无法生效。

## 可能原因
1. 网络路径的视频文件获取时长超时（默认10秒太短）
2. 元数据拉取失败但没有正确显示错误

## 修复方案

### 1. 增加网络路径超时时间
```python
# 网络路径需要更长的超时时间
timeout = 30 if video_path.startswith('\\\\') or video_path.startswith('//') else 10
```

### 2. 添加更详细的日志
```python
logger.info(f"获取时长成功: {video_path} -> {duration}秒")
```

## 使用方法

1. **扫描目录**：点击"选择目录"按钮
2. **扫描完成**：弹出确认对话框
3. **选择"是"**：自动拉取元数据
4. **查看日志**：检查terminal中的日志输出

## 测试结果
✅ 元数据提取器导入成功
✅ 已增加网络路径超时时间到30秒
