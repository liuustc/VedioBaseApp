# 封面提取改进

## 问题
双击打开详情后，封面都是一团漆黑，因为：
1. 默认提取第1秒，但很多电影开头是黑屏
2. 只尝试一个时间点，失败后没有备选方案

## 解决方案
改进 `metadata.py` 中的 `extract_cover()` 方法：

### 改进内容
1. **尝试多个时间点**：1秒、5秒、10秒、30秒、60秒
2. **检查文件大小**：避免提取到黑屏（文件大小小于1KB视为黑屏）
3. **自动重试**：一个时间点失败后自动尝试下一个

### 代码逻辑
```python
timestamps_to_try = [timestamp, 5.0, 10.0, 30.0, 60.0]

for ts in timestamps_to_try:
    # 提取封面
    # 检查文件大小
    if file_size > 1000:  # 大于1KB
        return 封面路径
    else:
        # 文件太小，可能是黑屏，继续尝试下一个时间点
        continue
```

## 使用方法
无需修改，程序会自动：
1. 首先尝试提取指定时间点（默认1秒）
2. 如果文件太小（黑屏），尝试5秒
3. 继续尝试10秒、30秒、60秒
4. 如果都失败，返回None
