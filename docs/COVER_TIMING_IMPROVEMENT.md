# 封面提取时间点改进

## 问题
之前的时间点（5秒、10秒、15秒）太靠前，大部分是片头部分。

## 改进方案

### 单张封面提取（extract_cover）
**之前：** `[1.0, 5.0, 10.0, 30.0, 60.0]` 秒
**现在：** `[timestamp, 300.0, 900.0, 1800.0, 2700.0]` 秒
- 5分钟（300秒）
- 15分钟（900秒）
- 30分钟（1800秒）
- 45分钟（2700秒）

### 多张封面提取（extract_multiple_covers）
**之前：** `[5.0, 15.0, 30.0]` 秒
**现在：** `[1800.0, 2700.0, 3600.0]` 秒（30分钟、45分钟、60分钟）

### 智能调整逻辑
1. **视频时长 < 3分钟**：使用 `[30.0, 60.0, 90.0]` 秒
2. **视频时长 < 5分钟**：使用 `[60.0, 120.0, 180.0]` 秒
3. **视频时长 >= 5分钟**：使用 `[1800.0, 2700.0, 3600.0]` 秒
4. **自动调整**：如果目标时间点超出视频范围，自动调整到有效范围内

## 时间点选择理由

### 为什么选择30分钟、45分钟、60分钟？
1. **避开片头**：大部分电影片头在5-10分钟内结束
2. **避开片尾**：结尾通常有字幕和彩蛋
3. **覆盖主要内容**：这些时间点通常在电影的高潮部分
4. **均匀分布**：覆盖电影的前、中、后三个阶段

### 对于短视频的处理
- **< 3分钟**：使用30秒、60秒、90秒
- **3-5分钟**：使用60秒、120秒、180秒
- **> 5分钟**：使用30分钟、45分钟、60分钟
