# 表头排序功能

## 功能描述
点击表格的表头时，按点击的那一栏排序，再点击一次则反向排序。

## 实现细节

### 1. 启用排序
```python
self.setSortingEnabled(True)
```

### 2. 连接表头点击信号
```python
self.horizontalHeader().sectionClicked.connect(self.on_header_clicked)
```

### 3. 排序逻辑
- 点击同一列：切换排序顺序（升序/降序）
- 点击不同列：按新列排序，升序开始

### 4. 排序键选择
- **ID列**：按数字排序
- **标题列**：按字符串排序（不区分大小写）
- **文件大小列**：按原始字节数排序
- **时长列**：按秒数排序
- **评分列**：按评分值排序

### 5. 数据存储
- 保存原始电影数据到 `self.movies_data`
- 使用 `Qt.ItemDataRole.UserRole` 存储原始值用于排序

## 使用方法

1. **点击表头**：按该列排序
2. **再次点击同一表头**：反向排序
3. **点击不同表头**：按新列排序
